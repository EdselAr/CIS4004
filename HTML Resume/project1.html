University Personnel Management System

Project Description

For my final project in COP 3330 (Object Oriented Programming), I developed a comprehensive Java application to manage university records. The screenshot above demonstrates the console-based menu system where users can add faculty, print tuition invoices, and manage staff records.

The program utilizes advanced Object-Oriented principles, specifically inheritance and abstraction, to handle different personnel types under a unified system.

Key Features & Technologies

  • Language: Java (JDK 17+)
  • Core Concepts: Inheritance, Abstract Classes, Polymorphism, and Encapsulation.
  • Functionality: As shown in the image, the system handles invalid inputs (e.g., "No student matched") and retrieves faculty details like Department and Rank.
